# varpred
varpred is a set of tools to predict molecular phenotype by variation,which include a machine learning model created by GBDT.

## Requirements
Package (python 3.6+):
    numpy (>=1.17.0)
    pandas (>=0.20.3)
    matplotlib (>=3.2.2)
    scikit-learn (>=0.20.3)
    lightgbm (>=3.1.0)
    shap (>=0.36.0)
